    In Go, every program is part of a package.
    We define this using the package keyword.
    In this example, the program belongs to the main package.

Line2:  -> ( import )
    import ("fmt") lets us import files included in the fmt package.

Line3:  -> ( func )
    func main() {} is a function. Any code inside its curly brackets {} will be executed.

    fmt.Println() is a function made available from the fmt package. It is used to output/print text.
